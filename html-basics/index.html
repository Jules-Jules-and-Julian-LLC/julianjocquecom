How to Code HTML

The basics of building web pages by hand.
No AI required - just you and a text editor.

What is HTML?

HTML stands for HyperText Markup Language. It's how you tell a web browser what to show on a page. You wrap text in "tags" to give it meaning and structure.

Tags look like this: <tagname> to open, and </tagname> to close.

The Simplest Web Page

Here's the bare minimum HTML you need for a working web page:

<!DOCTYPE html> <html> <head> <title>My Page</title> </head> <body> <h1>Hello World</h1> <p>This is my first web page!</p> </body> </html>
Try it: Copy this into a file called index.html, save it, and double-click to open in your browser.

Basic Tags You'll Use

Text & Headings

<h1>...</h1> Biggest heading (page title)
<h2>...</h2> Second-level heading
<h3>...</h3> Third-level heading (and so on to h6)
<p>...</p> Paragraph of text
<br> Line break (no closing tag needed)
<strong>...</strong> Bold text
<em>...</em> Italic text

Links & Images

<a href="url">...</a> A clickable link
<img src="photo.jpg"> An image (no closing tag)

Containers

<div>...</div> A box/section (for grouping things)
<span>...</span> Inline container (for styling bits of text)

Example: A Link

<a href="https://google.com">Click here to visit Google</a>
Result
Click here to visit Google

Example: An Image

<img src="photo.jpg" alt="A description of the image">
Note: The alt attribute describes the image for screen readers and shows if the image fails to load.

Adding Style with CSS

CSS (Cascading Style Sheets) controls how your HTML looks - colors, fonts, sizes, spacing. You add it inside a <style> tag in the <head>.

<!DOCTYPE html> <html> <head> <title>Styled Page</title> <style> /* This makes the whole page have a light background */ body { background-color: #f5f5f5; font-family: Arial, sans-serif; } /* This makes h1 headings blue */ h1 { color: navy; text-align: center; } /* This adds padding around paragraphs */ p { padding: 20px; line-height: 1.6; } </style> </head> <body> <h1>Welcome!</h1> <p>Now my page has some style.</p> </body> </html>

Common CSS Properties

Colors & Backgrounds

color: blue; Text color (use names or hex codes like #ff0000)
background-color: #fff; Background color

Text & Fonts

font-size: 20px; How big the text is
font-family: Arial; What font to use
font-weight: bold; Make text bold
text-align: center; Center, left, or right align

Spacing

padding: 20px; Space INSIDE the element
margin: 20px; Space OUTSIDE the element

Borders & Boxes

border: 1px solid black; A line around the element
border-radius: 10px; Rounded corners
width: 300px; How wide something is
max-width: 600px; Maximum width (won't grow past this)

Styling Specific Things

To style only certain elements, give them a class and target that class in CSS with a dot:

<style> .highlight { background-color: yellow; padding: 5px; } .big-text { font-size: 24px; } </style> <p class="highlight">This paragraph has a yellow background.</p> <p class="big-text">This paragraph is bigger.</p> <p class="highlight big-text">This has both!</p>

Your Workflow

  1. Open your site/hand_coded/index.html in a text editor (Notepad, TextEdit, or VS Code)
  2. Make changes to the HTML or CSS
  3. Save the file (Ctrl+S on Windows, Cmd+S on Mac)
  4. Refresh your browser to see the changes
  5. Repeat until it looks how you want
Common Mistakes:
  • Forgetting to close tags (</p>, </div>, etc.)
  • Missing semicolons in CSS (color: blue; not color: blue)
  • Typos in property names (it's background-color, not backgroud-color)

What's Next?

Once you're comfortable editing HTML, you have choices:

  • Keep hand-coding - look things up online when you want to do something new
  • Use AI for help - describe what you want and learn from what it gives you
  • Mix both - use AI to generate a starting point, then hand-edit to perfect it
