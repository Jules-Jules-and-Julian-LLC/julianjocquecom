The Art of Vibe Coding

You don't need to know code. You just need to know what you want.

Step 1: The Request

Go to ChatGPT, Gemini, or Claude. Treat the AI like a contractor. Be specific.

BAD PROMPT: "Make my site cool."

GOOD PROMPT: "I have an HTML file. I want to add a music player that plays jazz loops. Make the player look like an old cassette tape. Write the full HTML and CSS for me."

Step 2: The Copy/Paste

The AI will spit out a block of code.

  1. Copy that code.
  2. Open your index.html file in Notepad or VS Code.
  3. Delete everything (yes, really).
  4. Paste the new code.
  5. Save.

Step 3: The Check

Double-click your index.html file to open it in Chrome. Does it look right?

  • Yes? Go to the Publishing Guide.
  • No? Tell the AI: "That looks broken. The play button is too small. Fix it."
