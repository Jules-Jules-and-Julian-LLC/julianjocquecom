How to Vibe Code

Making websites without knowing how to code.
You describe what you want, the AI builds it.

1

Open an AI Chat

Pick one of these AI assistants (they all work similarly):

Google Gemini ChatGPT Claude
2

Give It These Instructions First

Before asking for anything, copy and paste this message to teach the AI how to help you:

Copy this first You are helping me build a simple website. I don't know how to code. Rules: - Give me ONE complete file called index.html - Put all the styling inside the file (no separate files) - Put any interactive stuff inside the file too - Don't explain the code, just give me the file - Make it look nice on phones too
3

Describe What You Want

Now just tell the AI what you're imagining. Be specific about colors, feelings, and what should happen when someone visits.

"I want a page with my name in big letters at the top. The background should be a calm blue color. Below my name, add a photo placeholder and a short paragraph about me. When someone hovers over my name, it should glow a little."
Tip: The more details you give, the better it works. Mention colors, feelings, what things should do when clicked, etc.
4

Save the Code

The AI will give you a big block of code. Here's what to do:

  1. Select ALL the code the AI gives you and copy it
  2. Open your site folder on your computer
  3. Open the file called index.html (right-click → Open With → Notepad or TextEdit)
  4. Select everything in that file and delete it
  5. Paste the new code
  6. Save the file (Ctrl+S on Windows, Cmd+S on Mac)
5

Preview It

Double-click the index.html file. It will open in your web browser so you can see what it looks like.

Not quite right? Go back to the AI and say something like: "The text is too small" or "Make the background darker" or "That looks broken, please fix it." Keep refining until you're happy!
6

Publish It

Once it looks good, upload it to make it live on the internet:

Go to Publishing Guide →

That's it!

You just made a website without writing any code yourself.
Keep experimenting and asking the AI for changes until it's perfect.
