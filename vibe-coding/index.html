The Manual

"Vibe coding lets you make a website that does something you know a website can do, even if you don't know how to make it do that."

The Secret

You do not need to learn HTML. You just need to be able to describe what you want.

If you can say: "I want a button that makes a fart sound," you can build it.

Step 1: The Request

Go to ChatGPT, Gemini, or Claude. Treat the AI like a contractor. Be specific.

PROMPT: "I have an HTML file. I want to add a music player that plays lo-fi beats. Make the player look like an old cassette tape. Write the full HTML and CSS for me."

Step 2: The Copy/Paste

The AI will give you code.

  1. Copy that code.
  2. Open your index.html file on your computer.
  3. Delete everything in the file. (Yes, really).
  4. Paste the new code.
  5. Save.

Step 3: Test It

Double-click the file to open it in Chrome. Does it work?

If yes: Go to the Upload Guide to publish it.
If no: Tell the AI "That looks broken, fix it."
