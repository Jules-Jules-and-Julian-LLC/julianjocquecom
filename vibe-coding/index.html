The Manual

"Vibe coding lets you make a website that does something you know a website can do, even if you don't know how to make it do that."

Step 1: Train the AI

Open Gemini (or ChatGPT). Before you ask for anything, Copy and Paste this "System Instruction" into the chat. This teaches the AI how to code for you.

Copy this block:
You are an expert web developer for a user who does not write code. RULES: 1. Output a SINGLE `index.html` file. 2. Include all CSS inside <style> tags. 3. Include all JavaScript inside <script> tags. 4. If I ask for interactivity (clicking, sounds, animations), use modern JavaScript. 5. Do not explain the code. just give me the full file.

Step 2: The Request

Now, ask for what you actually want. Be specific about the "Vibe."

EXAMPLE PROMPT: "I want to add a button in the center that says 'Do Not Press'. When I click it, I want a confetti explosion animation and a sound effect. Make the background red."

Step 3: The Copy/Paste

The AI will give you a big block of code.

  1. Copy that code.
  2. Open your index.html file on your computer.
  3. Delete everything in the file. (Yes, really).
  4. Paste the new code.
  5. Save.

Step 4: Test & Publish

Double-click the file to open it in Chrome. Does it work?

If yes: Go to the Upload Guide.
If no: Tell the AI "That looks broken, fix it."
